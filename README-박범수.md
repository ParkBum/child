# 프로젝트명 : 어린이 집마련 프로젝트

구디마지막 프로젝트_어집프

담당 업무

- D3.js 그래프 시각화 
- 지도 시각화        
- 회원 가입 및 로그인 (백/프론트) 
- 로그인 API 
- 주소 API
- Git 관리

사용된 언어 및 기술들 
HTML, CSS, JAVASCRIPT, Jquery, Ajax, Java, Jsp, SpringFramework, Json/TopoJson, MariaDB, JSTL, EL, MyBatis

 
개요 : 서울 내, 어린이집에 대한 위치정보와 상세정보를 한눈에 제공하고 실질적인 후기를 공유하여 
      부모님들이 어린이집을 분별하여 선택할 수 있는 정보를 제공해주기 위한 홈페이지 제작
